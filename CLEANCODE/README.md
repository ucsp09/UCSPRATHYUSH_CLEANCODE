# Running
In the terminal go the path where folder CLEANCODE exists.
## Windows
        Path: C:\Users\Username\Desktop\CLEANCODE
        Compiling: javac -cp . src\*.java test\*.java
        Execution: java -cp . test.Client
## Mac/Linux
        Path: C:/Users/Username/Desktop/CLEANCODE
        Compiling: javac -cp . src/*.java test/*.java
        Execution: java -cp . test.Client
